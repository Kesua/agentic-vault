# Ad-hoc Design Expert

This skill has been split into three topic-based skills because the merged bundle was too large to be practical as a single skill.

Use:
- `adhoc_design_foundation` for context gathering, design direction, onboarding, extraction, distillation, and clarification
- `adhoc_design_build` for implementation, layout, color, typography, motion, adaptation, and visual expansion
- `adhoc_design_review` for critique, audit, polish, hardening, optimization, normalization, and tone-down passes

## Vault Integration Notes
- Follow this vault's `AGENTS.md`, `SOUL.md`, and `MEMORY.md` first.
- Inspect relevant local notes before design work, especially project notes under `30_Projects/`.
- Persist durable design context in the relevant project note or project snapshot instead of `.impeccable.md` or other external tool-specific files.
