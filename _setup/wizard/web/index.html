Connect Slack

Step 1: Create a Slack App

  1. Open Slack App Management
  2. Click "Create New App" > "From an app manifest"
  3. Pick your workspace
  4. Paste this YAML manifest:
display_information:
  name: Vault Slack Read
features:
  bot_user:
    display_name: Vault Slack Read
    always_online: false
oauth_config:
  scopes:
    bot:
      - channels:history
      - channels:read
      - groups:history
      - groups:read
      - im:history
      - im:read
      - mpim:history
      - mpim:read
      - users:read
      - users:read.email
      - chat:write
      - chat:write.public
      - im:write
      - mpim:write
      - users:read
settings:
  interactivity:
    is_enabled: false
  org_deploy_enabled: false
  socket_mode_enabled: false
  token_rotation_enabled: true

Step 2: Install to Workspace

In the app settings, go to "OAuth & Permissions" and click "Install to Workspace". Approve the requested permissions.

Step 3: Paste Bot Token

After installing, copy the "Bot User OAuth Token" from the OAuth page.

A short name for this workspace (e.g. "private", "work")
